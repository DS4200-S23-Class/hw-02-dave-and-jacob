Data Types

UFO

  • Row Data Type: Item (Rows in tables are individual entities that are discrete
  • Datetime: Attribute
    • Attribute Type: Ordered (Sequential Quantitative because they can be ordered and continually incraese)
  • City: Attribute
    • Attribute Type: Categorical (Specific attribute that can be observed, can be grouped into categories)
  • State: Attribute
    • Attribute Type: Categorical (Can be observed and grouped into categories)
  • Country: Attribute
    • Attribute Type: Categorical (Can be observed and grouped into categories)
  • Duration(seconds): Attribute
    • Attribute Type: Ordered (Quantitative Sequential because they start at 0 and can only increase)

Aiports

  • Row Data Type: Item
  • AIRPORT_NAME: Attribute
    • Attribute Type: Categorical (Categorical because can be obeserved and grouped into categories)
  • LATITUDE & LONGITUDE: Position (Provides a location in 2d, coordinates)

Google-Web

  • Row Data Type: Link (Shows the relationship between two nodes)
  • FromNodeId: Attribute
    • Attribute Type: Categorical (Although numeric, its used as a unique identifier rather than ordering values)
  • ToNodeId: Attribute
    • Attribute Type: Categorical (Although numeric, its used as a unique identifier rather than ordering values)
